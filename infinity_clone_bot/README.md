# Infinity Clone Bot

Клон Telegram-бота `infinity_exchange_bot` на базе собранного флоу из `output/infinity_exchange_bot` с добавленной админкой.

## Что есть

- Основной флоу с медиа и кнопками:
  - `Купить`
  - `Продать`
  - `Командный модуль`
  - `FAQ`
  - `Контакты`
- Авторасчёт в сценариях покупки/продажи по курсам криптовалют.
- Курсы подтягиваются с CoinGecko (`BTC`, `LTC`, `USDT`, в `RUB`), есть fallback.
- Админка `/admin`:
  - смена комиссии (%)
  - управление реквизитами (режим, реквизиты, банк, способы оплаты: добавить/удалить)
  - смена всех ссылок (faq/channel/chat/reviews/review_form/manager/operator/terms)
  - редактирование любых переменных `.env` (`KEY=VALUE`)
  - ручное обновление курсов.
- Данные хранятся локально:
  - `data/settings.json` (комиссия + ссылки + реквизиты/способы оплаты)
  - `data/users.json` (примитивный профиль/история).

## Структура

- `main.py` — точка входа.
- `app/config.py` — загрузка `.env`.
- `app/storage.py` — локальное хранилище JSON.
- `app/rates.py` — курсы.
- `app/handlers/*.py` — роутеры по модулям.
- `assets/*.jpg` — медиа из собранного флоу.

## Запуск

1. Установить зависимости:
   - `pip install -r requirements.txt`
2. Подготовить env:
   - `copy .env.example .env`
   - заполнить `BOT_TOKEN` и `ADMIN_IDS`
3. Запуск:
   - `python main.py`

### Hot Reload (dev)

- Автоперезапуск при изменении `*.py` и `.env`:
  - `python dev.py`

## Команды

- `/start` — главное меню.
- `/rates` — показать текущие курсы.
- `/admin` — админка (только `ADMIN_IDS`).

В админке изменения `.env` применяются сразу.  
Если изменён `BOT_TOKEN`, бот автоматически перезапускается.

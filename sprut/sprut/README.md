# Telegram Bot с капчей

Бот на aiogram, который при команде /start выдает капчу с выбором эмодзи.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Получите токен бота у @BotFather в Telegram

3. Создайте файл `.env` и укажите в нем ваш токен:
```
BOT_TOKEN=ваш_токен_здесь
```

4. Запустите бота:
```bash
python bot.py
```
